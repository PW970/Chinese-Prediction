技术实现说明

核心公式
系统采用标准小六壬起卦法。取农历月、日、时之和，减去偏移量，通过取模运算确定六神（大安、留连、速喜、赤口、小吉、空亡）的归属。

时间处理
系统集成第三方历法插件，自动处理节气、闰月及二十四小时制与十二时辰的转换关系，确保测算基准时间的严谨性。

数据分离
为了便于后续的国际化扩展或版本修订，所有文化层面的文本描述均独立存储于 JSON 数据文件中，与底层逻辑解耦。
